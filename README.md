# PeachTravels
This is a web-app that allows people to plan their travels and trips in a simple way and also can be shared with others.

Introduction: This is the first personal project that I have made after graduationg college. It's purpose is to be a travel planner so that people can organize their plans and be able to share them with others. It would also ideally include the predicted weather, and other stats for the area during that trip.
